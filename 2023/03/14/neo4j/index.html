1 总体概述以及 Neo4j 简介

本博客记录如何构建一个有关陶瓷的多模态知识图谱，所使用到的数据库平台是 neo4j。Neo4j 是使用 Java 编写的一个基于图的开源数据库，随着技术的发展，人们对数据的需求已经不仅限于数据本身，更多是数据与数据之间的联系。而如果用传统的数据库来存储连接的数据，会导致遍历大量数据时不能提供合适的性能。Neo4j 遵循图数据模型，使用 cypher 语言对数据库进行操作，且具有可视化界面来直观反应数据节点之间的联系。

1.1 Mac 版本安装（Apple Silicon/Intel 通用）

  1. 安装适配 Apple Silicon 的 jdk 版本（博主安装的是 Java17）。

  2. 从 neo4j 官网下载最新安装包，博主这里安装的是 neo4j-5.5.0，
    随附官网 community server 的下载页面 https://neo4j.com/download-center/#community。

  3. 下载后解压 tar.gz 文件（尽量不要解压在下载文件夹里，最好选择一个全英文路径且没有空格的路径下），终端进入/neo4j-community-5.5.0/bin 目录下，执行neo4j start命令。如果成功执行则可以在 http://localhost:7474/browser/ 本地网站访问，显示的页面如下图则表示正常安装。

2 陶瓷文物数据

2.1 数据来源——故宫博物馆网站

  • 藏品目录（部分）：https://www.dpm.org.cn/collection/ceramics.html
    本目录下的陶瓷文物共有 1,388 条记录，对应每个文物都具有以下（或部分）栏目：文物名称、文物对应图片、时代、分类（一般来说是制作工艺，如釉下彩、青瓷等）、窑口、简介、关键词（和该文物相关的内容，如青花、吉祥图案）等。是博主目前首选的数据采集内容。
  • 数字文物库：https://digicol.dpm.org.cn/?category=6
    本目录下的陶瓷文物有 10,624 条记录，对应每个文物具有以下（或部分）栏目：文物名称、文物对应图片、时代、相关推荐词条（其中可能包含物件类型、图案纹样、形制等）。记录较多，后续大概率会加入陶瓷知识图谱中。需要注意的是，该文物库与上述藏品目录（部分）非包含关系。
  • 数字多宝阁：https://www.dpm.org.cn/shuziwenwu.html?name=陶瓷
    本目录下的陶瓷文物有 60 条记录，需要注意的是，每个陶瓷文物都具有3D 效果！，但由于数据较少，暂未加入本次数据采集中。
